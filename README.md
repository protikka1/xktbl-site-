# xltbl | Professional Business Solutions 🚀

Welcome to the official repository for **xltbl**, a high-performance, mobile-first consulting platform. xltbl specializes in bridging the gap between business ideas and technical reality, focusing on store setup, digital marketing training, and professional technical consulting.

---

## 📱 Project Overview
This entire platform was developed using an **Android-first workflow**, proving that professional-grade web applications can be built, tested, and deployed entirely from a mobile device.

### Tech Stack
* **Editor:** [Acode](https://acode.app/) (Android)
* **Framework:** [W3.CSS](https://www.w3schools.com/w3css/) for responsive, lightweight design.
* **AI Collaborator:** Google Gemini (Code logic & Content strategy).
* **Forms:** Formspree (Serverless contact management).
* **Hosting:** Google Firebase / Netlify.

---

## ✨ Key Features
* **Google Meet Integration:** Seamlessly book video consultations.
* **Appointment Scheduling:** Built-in calendar logic for 30-min and 1-hour sessions.
* **Responsive Navigation:** Modern hamburger menu optimized for mobile users.
* **Contact Hub:** A floating multi-action menu for instant communication (Call, Meet, Email).
* **Legal Transparency:** Built-in Privacy and Terms modals for professional compliance.

---

## 🛠️ Development Workflow
To maintain this site on Android:
1.  **Code:** Edit `index.html` within the Acode editor.
2.  **Test:** Use the internal Acode server or mobile browser console.
3.  **Version Control:** Push changes to this GitHub repository.
4.  **Deploy:** Automatic deployment via GitHub Actions to Firebase/Netlify.
